2.2 Formal Definition
A Threshold Series is defined as a sequence of positive, diminishing terms aₙ, such that:
lim(n→∞) aₙ = 0, but ∑(n=1)^∞ aₙ = ∞
Each term individually vanishes at infinity, yet the total sum grows without bound.

This structure matters because it models processes that grow without appearing to, that change without disruption, and that give rise to form through subtlety.

### **A.1.1 Basic Threshold Series Definition**
A **Threshold Series** is defined as:
```

```

**Properties:**

2. **Collective Behavior**: `∑_{n=1}^∞ aₙ` may converge or diverge depending on (α, β)
3. **Soft Divergence**: When divergent, growth is slower than polynomial

### **A.1.2 Convergence Theorem (Rigorous Proof)**

**Theorem A.1 (Threshold Series Convergence):**
For the series `∑_{n=2}^∞ 1/(n^α (ln n)^β)`:

1. **If α > 1**: Series converges for all β ∈ ℝ
2. **If α < 1**: Series diverges for all β ∈ ℝ  
3. **If α = 1**:
   - **β > 1**: Series converges
   - **β ≤ 1**: Series diverges (soft divergence)    (Eq. A.2)

**Proof:**

*Case 1 (α > 1):*
Using integral test with ε = α - 1 > 0:
```
∫₂^∞ 1/(x^α (ln x)^β) dx = ∫₂^∞ 1/(x^{1+ε} (ln x)^β) dx    (Eq. A.3)
```
Since x^{1+ε} dominates (ln x)^β for any β, the integral converges.

*Case 2 (α < 1):*
For ε = 1 - α > 0:
```
∫₂^∞ 1/(x^α (ln x)^β) dx = ∫₂^∞ 1/(x^{1-ε} (ln x)^β) dx ≥ ∫₂^∞ 1/x^{1-ε} dx = ∞    (Eq. A.4)
```

*Case 3 (α = 1):*
```
∫₂^∞ 1/(x (ln x)^β) dx    (Eq. A.5)
```

With substitution u = ln x, du = dx/x:
```
∫_{ln 2}^∞ 1/u^β du    (Eq. A.6)
```

This converges if and only if β > 1. ∎
